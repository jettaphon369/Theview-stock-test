# TheView Stock Firebase Free v1

เวอร์ชันนี้ทำให้ข้อมูลตามบัญชีได้: เข้าด้วยอีเมล/รหัสเดิมบนเครื่องอื่น ข้อมูลจะซิงก์ผ่าน Firebase Firestore

## กฎสำคัญ
- ใช้ Firebase Spark Plan เท่านั้น
- ห้ามเปิด Billing
- ถ้าเกินโควตา ระบบจะหยุดชั่วคราว ไม่คิดเงิน

## วิธีตั้งค่า
1. สร้าง Firebase Project
2. เลือก Spark Plan (ฟรี — ห้ามเปิด Billing)
3. เปิด Authentication > Email/Password
4. เปิด Firestore Database
5. คัดลอก Firebase Config มาใส่ที่ `config/firebase-config.js`
6. **สำคัญ:** ไปที่ Firestore Database > Rules แล้ววางเนื้อหาจากไฟล์ `firestore.rules` ในโปรเจกต์นี้ แล้วกด Publish
   (ถ้าไม่ทำขั้นตอนนี้ ข้อมูลผู้ใช้ทุกคนจะเปิดโล่ง ใครก็อ่าน/แก้ได้)
7. อัปโหลดทั้งโฟลเดอร์ขึ้น Netlify/Vercel/GitHub Pages

## บัญชี
กดสมัครบัญชีใหม่จากหน้า Login ได้เลย แล้วใช้บัญชีเดิมเข้าอีกเครื่อง ข้อมูลจะตามไป
(เอาบัญชีทดสอบที่ฝังในหน้า login ออกแล้ว เพื่อความปลอดภัย)

## ข้อจำกัด
- ยังไม่ใช้ AI Vision เสียเงิน
- AI Assist เป็นระบบฟรีจากฐานข้อมูล/คำค้น
- รูปถูกเก็บใน Firestore เป็น Base64 — ตอนนี้ระบบย่อขนาดรูป (สูงสุด 640px) และบีบอัดคุณภาพ JPEG ~60% ให้อัตโนมัติก่อนบันทึก เพื่อไม่ให้ชนลิมิต 1MB/เอกสารและประหยัดโควตาฟรี
- รองรับใช้งาน offline เบื้องต้นผ่าน Service Worker (cache หน้าเว็บ/ไฟล์ static ไว้ ไม่รวมข้อมูล Firestore)

## แก้ไขล่าสุด (v1.1)
- ลบบัญชีทดสอบที่ฝังในหน้า login ออก
- เพิ่ม `firestore.rules` — ต้องนำไปตั้งค่าใน Firebase Console เอง (ดูขั้นตอนข้างบน)
- เพิ่มการย่อ/บีบอัดรูปอัตโนมัติก่อนเก็บใน Firestore
- เพิ่มไอคอน PWA (`assets/icon-192.png`, `assets/icon-512.png`) และเชื่อมกับ manifest
- เพิ่ม Service Worker สำหรับใช้งาน offline เบื้องต้น
- เพิ่มถังขยะ (Trash): กด "ลบสินค้า" จะย้ายเข้าถังขยะแทนการลบถาวรทันที กู้คืนได้จากหน้าโปรไฟล์ > 🗑️ ถังขยะ ต้องพิมพ์ยืนยันเพื่อลบถาวรจริงเท่านั้น
